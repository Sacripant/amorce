Convention de nommage des class CSS

Le system de nomenclature de class CSS utilisé est un dérivé du système BEM (Block__Element--modifier)
Il utilise l'element en prefixe qui est inspiré de la sémantique des balise HTML.
.element__block--modifier L'element (suffixe) est séparé par 2 underscores du nom du bloc (__).
Le nom du block peut être composé avec des tiret. Ex: main-nav.

Liste des préfixes element utilisés couremment

  • h : pour le titre principale du bloc
  • l : pour une liste de block
  • i : pour l'item d'une liste de block
  • a : pour le lien principal du block
  • img : image du block
  • metas : pour le wrapper listant les metas de l'item (date, auteur, catégories, …)

D'autres préfixes pourront être créés pour les besoins du projet

Exemples

Bloc de menu principal

		
<nav class="main-nav">
	<ul class="l__main_nav">
		<li class="i__main-nav">
			<a href="#" class="a__main-nav">Link</a>
		</li>
	</ul>
</nav>

Liste d'articles

Les préfixes exceptionellement peuvent être cumulés, essentiellement dans les boucles (listes de résultats)

		
<sections class="articles">
	<ul class="l__articles">
		<li class="i__articles">
			<h2 class="h__i__articles">Titre de l'article</h2>
			<div class="img__i__articles">
				<img src="#" alt="#" />
			</div>
			<div class="metas__i__articles">
				Auteur + Date
			</div>
		</li>
		<li>…
	</ul>
</sections>
{% endblock %}

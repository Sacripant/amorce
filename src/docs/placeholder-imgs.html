Placeholder images

Ce module permet d'utiliser des images de remplacement dans différentes proportions standards pour vos prototypes.
Les images sont au format SVG et se trouvent dans le dossier build/img/placeholder. Des images "décorateurs" libre de droits sont également disponibles pour thématiser vos images.

Images & formats disponibles

  • picture-1x1.svg format carré
  • picture-2x1.svg format 2*1 paysage
  • picture-2x3.svg format 2*3 portrait
  • picture-3x2.svg format 3*2 paysage
  • picture-3x4.svg format 3*4 portrait
  • picture-4*3.svg format 4*3 paysage

Themes disponibles

  • grey (arrière plan gris)
  • flower
  • food
  • music
  • town

Utilisation

utilisez une balise img standard pointant vers l'image svg du format désiré. Ajoutez comme attribut alt="placeholder theme" en remplaçant theme par le mot clef du thème désiré.
